# Project GYMER


### Installtion and run
1. download repo
```code
git clone URL
cd URL
```
2. Install Dependencies
```code
npm i
``` 
3. Setup .env
4. start server
```code
npm run start
```
### Application Dependencies

### .env Structure
```code
MongoDB_URL=
PORT=3000
Secret_KEY=
RAZORPAY_ID_KEY=
```
